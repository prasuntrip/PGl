# NGL App Clone (Anonymous Messaging Platform)

This is a clone of the popular NGL (Not Gonna Lie) app built using **Spring Boot**. The platform allows users to receive anonymous messages from others via a unique profile link.

## 📝 Features

- User registration & login 
- Create a unique link to receive messages
- Send anonymous messages to other users
- View messages received
- RESTful APIs built with Spring Boot
- Database integration (JPA/Hibernate)
- Optional frontend using Thymeleaf

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot, Spring MVC, Spring Data JPA
- **Database:**  MySQL
- **Security:** Spring Security 
- **Build Tool:** Maven
- **Others:** Lombok, Thymeleaf (optional for API docs)
